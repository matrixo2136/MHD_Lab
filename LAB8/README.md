# LAB8 - Agregacje i wydajność

To laboratorium korzysta ze wspólnego folderu danych:

```text
data_common/
```

Nie trzeba kopiować `Online_Retail_II.xlsx` lub `Online_Retail_II.csv` do folderu LAB8.

## Wymagany plik danych

W folderze głównym projektu powinien znajdować się jeden z plików:

```text
data_common/Online_Retail_II.xlsx
```

albo:

```text
data_common/Online_Retail_II.csv
```

## Uruchomienie

Z głównego folderu projektu:

```bash
python LAB8/lab8_aggregation_performance.py
```

## Mniejszy test na PythonAnywhere

Jeżeli PythonAnywhere ma problem z pamięcią:

```bash
LAB8_MULTIPLIER=2 python LAB8/lab8_aggregation_performance.py
```

## Wyniki

Wyniki zapisują się do:

```text
LAB8/output/aggregation_performance/
```
